## RunRhythm

RunRhythm is a React-based running and music discovery app that helps users track runs and find Spotify playlists based on mood and pace.

### Features
- User login and registration
- Track runs with time, distance, and mood
- Filter Spotify playlists by mood and pace
- Save favorite playlists and preview tracks
- View profile with run history and last played mood
